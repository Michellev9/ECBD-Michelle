## Descripción General

Este repositorio contiene las prácticas realizadas durante la materia de Extracción de Conocimiento en Bases de Datos, enfocadas en la obtención, limpieza, transformación, análisis y visualización de datos mediante herramientas de ciencia de datos.

## Objetivo

Aplicar técnicas de extracción y análisis de datos utilizando herramientas de ciencia de datos para transformar datos en información significativa, desarrollando habilidades en limpieza, exploración y visualización de datos mediante Python y Jupyter Notebook.

* Matplotlib
* Git
* GitHub

## Instrucciones de Ejecución General

1. Clonar el repositorio:

```bash
git clone https://github.com/Michellev9/ECBD-Michelle.git

```

2. Ingresar al proyecto:

```bash
cd ECBD-Michelle
```

3. Instalar las dependencias necesarias:

```bash
pip install pandas numpy matplotlib jupyter
```

4. Ejecutar Jupyter Notebook:

```bash
jupyter notebook
```

5. Abrir el laboratorio deseado y ejecutar las celdas correspondientes.
